# Game

Аркада на SpriteKit: уклоняйтесь от опасных снарядов, собирайте энергию и выживайте как можно дольше.

## Что добавлено
- Главное меню с запуском игры, переходом в настройки и выбором сложности.
- Режимы сложности: легко, нормально, сложно.
- Настройки звука и вибрации.
- Улучшенная визуальная стилистика (панели, кнопки, аура игрока, эффекты).
- Система уровней, рост скорости и опасности по ходу игры.
- Лучший счёт, отображаемый в меню.
